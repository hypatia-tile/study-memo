## Install `lua-language-server` on Ubuntu

[lua-language-server](https://github.com/LuaLS/lua-language-server) is a language server for Lua.
It provides features like auto-completion, shows diagnostics, and more.

### Installation Steps

There are several ways to install `lua-language-server` on Ubuntu.

I choose to install it by building from source.

```bash
$ sudo apt update
$ sudo apt install -y git cmake g++ libssl-dev
$ git clone --recurse-submodules https://github.com/LuaLS/lua-language-server.git
$ cd lua-language-server
$ ./make.sh
```

[luals.github.io wiki](https://luals.github.io/wiki/build/)
