## Install Alacritty terminal on Ubuntu

### Reference:
- https://zenn.dev/shinnopo/articles/798398b1d87f62

### 1. Clone the Alacritty repository


```sh
git clone https://github.com/alacritty/alacritty.git
cd alacritty
```

### 2. Install Rust and Cargo

```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```


### 3. Install dependencies

```sh
sudo apt install cmake pkg-config libfreetype6-dev libfontconfig1-dev libxcb-xfixes0-dev python3
```

### 4. Build Alacritty

```sh
cargo build --release
```

### 5. After building, check terminfo

```sh
infocmp alacritty
```

- What is `infocmp`?
- What is `terminfo`?

### 6. If you got error in Step 5. Do the following:


```sh
sudo tic -xe alacritty,alacritty-direct extra/alacritty.info
```

### 7. Configuration for Desctop Entry

```sh
sudo cp target/release/alacritty /usr/local/bin
sudo cp extra/logo/alacritty-term.svg /usr/share/pixmaps/Alacritty.svg
sudo desktop-file-install extra/linux/Alacritty.desktop
sudo update-desktop-database
```

### 8. Install man page


```sh
sudo mkdir -p /usr/local/share/man/man1
gzip -c extra/alacritty.man | sudo tee /usr/local/share/man/man1/alacritty.1.gz > /dev/null
```

### 9. Configuration for completion

```sh
echo "source $(pwd)/extra/completions/alacritty.bash" >> ~/.bashrc
```

## Use Alacritty as default terminal

You may want to open Alacritty terminal when you press `Ctrl + Alt + t`:


```sh
sudo update-alternatives --install /usr/bin/x-terminal-emulator x-terminal-emulator /usr/local/bin/alacritty 50
```

We can check priority of terminal emulators:

```sh
sudo update-alternatives --display x-terminal-emulator
```
