## Install Nerd Fonts on Ubuntu and reflect it.


### References:
- https://qiita.com/Mike3906/items/3ed4137f915465f957b3

### Steps:

#### 1. Download Nerd Fonts from the official website
  - https://www.nerdfonts.com/font-downloads


#### 2. Unzip the download file

```sh
filename="JetBrainsMono.zip"
extension="${filename##*.}"
filename="${filename%.*}"
mkdir ${filename} && pushd ${filename}
unzip ../${filename}.${extension}
popd
```

#### 3. Create the font directory if it does not exists

```sh
mkdir -p ~/.fonts
mv ${filename} ~/.fonts/
```

#### 4. Refresh the font cache

```sh
fc-cache -fv
```
