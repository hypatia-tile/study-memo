# Tips: update Git version

You may want to ensure you have installed the latest version of Git.
See [Download for Linux and Unix](https://git-scm.com/downloads/linux) to update Git version.
In this website, you can find what is the latest version of Git.
Anyway, the best way to keep your Git version up-to-date is to use a package manager like `brew` or `apt-get`.
